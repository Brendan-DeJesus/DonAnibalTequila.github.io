1. About Royalwine

Royalwine is a HTML5 Template built using Bootstrap framword and HTML5 and CSS3 technology suilable for wine shop, bar, restaurant website . It's compatible with all major web browsers and works nicely on smartphones, tablet, PCs and desktops.

Features

  • - 22 Home pages
  • - HTML5 and CSS3
  • - Bootstrap 3.x
  • - Responsive design
  • - Browser Compatibility
  • - Well Documented
  • - Clean code
  • - W3C Valid
  • - Easy to Customization
  • - Google Fonts
  • - Google Map
  • - Awesome font icons

1. Open link https://console.developers.google.com/flows/enableapi?apiid=webmasters&credential=client_key 

2. Press "Continue" to create project 

3. Press "go to credentials" 

4. On the left you click credentials -> click create credentials-> on top downlown you select "API key" 1 popub display you click close

5. copy API paste into "Your key"
